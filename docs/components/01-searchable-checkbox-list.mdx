## Props

| Name          | Type                          | Default               | Description                           |
| ------------- | ----------------------------- | --------------------- | ------------------------------------- |
| checkboxes    | `T[]`                         | -                     | Array of Checkbox                     |
| buttonLabel   | `string`                      | `"Validate changes"`  | Array of Checkbox                     |
| onClickButton | `((checkboxes: T[]) => void)` | -                     | Callback function of button `onClick` |
| placeholder   | `string`                      | `"Search in options"` | Array of Checkbox                     |

## ICheckbox

| Attribute          | Type               | Description    |
| ------------------ | ------------------ | -------------- |
| active             | `boolean`          | Checked or not |
| id <Required/>     | `number \| string` | id of checkbox |
| label: <Required/> | 'string'           | value of label |

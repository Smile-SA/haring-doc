## Props

| Name                | Type                                                                            | Default | Description                                                                                                                                               |
| ------------------- | ------------------------------------------------------------------------------- | ------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| collapseButtonProps | `Omit<ICollapseButtonProps, 'opened>`                                           | -       | Props, or function returning props for the [Controlled CollapseButton component](./collapse-button#controlled-1), with the exception of the `opened` prop |
| defaultOpenedIds    | `(number \| string)[]`                                                          |         | Default opened menus, using `id` field of `IMenuItem`                                                                                                     |
| defaultSelectedId   | `number \| string`                                                              |         | Default selected menu id, referencing the `id` field of `IMenuItem` in `menu`                                                                             |
| hasOnlyOneOpenMenu  | `boolean`                                                                       | `false` | Keeps only one menu per level open at once                                                                                                                |
| menu <Required/>    | `IMenuItem[]`                                                                   | -       | The `IMenuItem` list to be rendered                                                                                                                       |
| menuOpenValue       | `(number \| string)[]`                                                          |         | Controlled state of which menus are currently open, using `id` field of `IMenuItem` in `menu`                                                             |
| onMenuOpen          | `(id: number \| string, isOpened: boolean, path: (number \| string)[]) => void` |         | Callback fired when menu is opened or closed, `id` and `path` use the `id`s fields of `IMenuItem` in `menu`                                               |
| ...                 | -                                                                               | -       | extends [Paper props](https://v6.mantine.dev/core/paper/?t=props)                                                                                         |

### `IMenuItem`

| Attribute         | Type               | Description                   |
| ----------------- | ------------------ | ----------------------------- |
| children          | `IMenuItem[]`      | Nested array of menus         |
| content           | `ReactNode`        | Content shown inside the menu |
| id <Required/>    | `number \| string` | Menu id                       |
| label <Required/> | `number \| string` | Menu label                    |
| leftIcon          | `ReactNode`        | Menu icon                     |

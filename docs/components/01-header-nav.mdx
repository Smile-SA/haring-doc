## Props

| Name              | Type                  | Default | Description                               |
| ----------------- | --------------------- | ------- | ----------------------------------------- |
| menus <required/> | `IHeaderNavMenu<T>[]` | -       | Array of element of type `IHeaderNavMenu` |
| isMobile          | `boolean`             | `false` | Define display for mobile or computer     |
| navLinkComponent  | `ElementType`         | `"a"`   | Define balise type of navLink             |

## IHeaderNavMenu

`IHeaderNavMenu` has a generic `T` that extend of `number | string` and it extend of `IMenuItem<T>`

| Name     | Type                  | Description                         |
| -------- | --------------------- | ----------------------------------- |
| children | `IHeaderNavMenu<T>[]` | Item array of type `IHeaderNavMenu` |
| url      | `string`              | Source url                          |

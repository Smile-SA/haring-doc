## Props

| Name                    | Type                                                         | Default             | Description                                                                                                                                                          |
| ----------------------- | ------------------------------------------------------------ | ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| children <Required/>    | `ReactNode`                                                  | -                   | Content rendered below the tabs                                                                                                                                      |
| tabs <Required/>        | `ReactElement<HTMLButtonElement>[]`                          | -                   | Array of `<Tabs.Tab>` components, see [Tabs Usage](https://v6.mantine.dev/core/tabs/)                                                                                |
| dropdownButtonAriaLabel | `string`                                                     | `"Overflow Button"` | Aria label of the dropdown button                                                                                                                                    |
| dropdownButtonProps     | [`ButtonProps`](https://v6.mantine.dev/core/button/?t=props) | -                   | Props for the [Mantine Button](https://v6.mantine.dev/core/button/?t=props) component                                                                                |
| tabsListProps           | [`TabsListProps`](https://v6.mantine.dev/core/tabs/?t=props) | -                   | Props for the [Mantine Tabs.List](https://v6.mantine.dev/core/tabs/?t=props#tabslist-component-props) component, see [Tabs Usage](https://v6.mantine.dev/core/tabs/) |
| ...                     | -                                                            | -                   | extends [Tabs props](https://v6.mantine.dev/core/tabs/?t=props)                                                                                                      |

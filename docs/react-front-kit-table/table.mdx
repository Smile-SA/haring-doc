## Props

| Name            | Type                                                                          | Default           | Description                                                                                                                                                           |
| --------------- | ----------------------------------------------------------------------------- | ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| actions         | [`ITableAction`](#itableaction)                                               | `[]`              | Array of [`IAction`](../shared-types/actions#iactiont)s for the table                                                                                                 |
| menuLabel       | `string`                                                                      | `"Other actions"` | Tooltip label of the "`...`" extra actions menu                                                                                                                       |
| paginationProps | [`Partial<IPaginationProps>`](../react-front-kit/components/pagination#props) | -                 | Props of [Pagination](../react-front-kit/components/pagination#props) component                                                                                       |
| rowActionNumber | `number`                                                                      | `0`               | Index that determines where the row `actions` array in the actions column is sliced, all actions after this index will be moved inside the "`...`" extra actions menu |
| ...             | -                                                                             | -                 | extends [MRT_TableOptions props](https://www.mantine-react-table.com/docs/api/table-options)                                                                          |

### `ITableData<Data extends Record<string, unknown>`

Corresponds to either a `Data` object or an array of `Data` objects, or [`MRT_Row`](https://www.mantine-react-table.com/docs/guides/customize-components#mantine-props-and-types)`<Data>` or [`MRT_Row`](https://www.mantine-react-table.com/docs/guides/customize-components#mantine-props-and-types)`<Data>[]`

[`MRT_Row`](https://www.mantine-react-table.com/docs/guides/customize-components#mantine-props-and-types) is a [MantineReactTable](https://www.mantine-react-table.com/docs/getting-started/usage) wrapper type that correspond to a table row.

### `ITableAction`

Corresponds to [`IAction`](../shared-types/actions#iactiont)[`<ITableData>`](#itabledata)

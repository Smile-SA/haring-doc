## Props

| Name            | Type                                                                                                                                                                                                | Default                               | Description                                                               |
| --------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------- | ------------------------------------------------------------------------- |
| browseLabel     | `string`                                                                                                                                                                                            | `"Browse your device"`                | String displayed at the bottom next to the Eye icon                       |
| children        | `ReactElement`                                                                                                                                                                                      | -                                     | Element inserted at the bottom of the container                           |
| dragLabel       | `string`                                                                                                                                                                                            | `"Drag and drop your documents here"` | String displayed in the center of the container                           |
| files           | [`IFile[]`](#ifile)                                                                                                                                                                                 | `[]`                                  | Controlled value of the current files                                     |
| buttonPlusTitle | `string`                                                                                                                                                                                            | `"Add button"`                        | Tooltip title of top `+` button                                           |
| onDrop          | `(files: FileWithPath[]) => void` <br/><br/> See [`FileWithPath`](https://github.com/rtivital/react-dropzone-esm/blob/4bbb571f0d4c01d65025165d891a94c012a468fe/types/react-dropzone.d.ts#L3C1-L3C1) | -                                     | Called when valid files are dropped into dropzone                         |
| onRemoveFile    | [`(file: IFile) => void`](#ifile)                                                                                                                                                                   | -                                     | Called when added file is removed by clicking on `x` button               |
| ...             | -                                                                                                                                                                                                   | -                                     | extends [Dropzone props](https://v6.mantine.dev/others/dropzone/?t=props) |
